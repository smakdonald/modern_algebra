Example 17.3. \(\Aut(\C/\R)\).

I claim \(\Aut(\C/\R)\) has two elements (and so is a cyclic group of order \(2\)): the identity map on \(\C\) and the element \(\s\) given as complex conjugation. Each of these is an element of \(\Aut(\C/\R)\) â€” for \(\s\text{,}\) this amounts to the fact that complex conjugation commutes with addition and multiplication of complex numbers (and that it sends \(1\) to \(1\)).
To see these are the only elements of \(\Aut(\C/\R)\text{,}\) suppose \(\tau \in \Aut(\C/\R)\text{.}\) For any \(z = a + ib \in \C\text{,}\) we have \(\tau(z) = a + b \tau(i)\) since \(\tau|_\R = \id_\R\text{.}\) Moreover, \(-1 = \tau(-1) = \tau(i \cdot i) = \tau(i) \cdot \tau(i)\) and so \(\tau(i) = \pm 1\text{.}\) Thus \(\tau = \id\) or \(\tau = \s\text{.}\)
