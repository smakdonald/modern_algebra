Theorem 13.52. Classification of Finite Dimensional Vector Spaces.

Let \(F\) be a field.
  1. Every finite dimensional vector space \(V\) over \(F\) is isomorphic to \(F^{n}\) for \(n=\operatorname{dim}_{F}(V)\text{.}\)
  2. For any \(m, n \in \mathbb{Z}_{\geqslant 0}, F^{m} \cong F^{n}\) if and only if \(m=n\text{.}\)

Proof.

  1. Let \(V\) be a finite dimensional \(F\)-vector space. Then \(F\) has a finite spanning set \(S\) and by TheoremÂ 13.40 there is a basis \(B \subseteq S\) for \(V\text{.}\) Notice that \(B\) is necessarily finite and \(V=F B\text{.}\) Set \(|B|=n\) and \(B=\left\{b_{1}, \ldots, b_{n}\right\}\text{.}\) By the UMP for Free \(R\)-Modules, there is a linear transformation \(f: F^{n} \rightarrow V\) such that \(f\left(e_{i}\right)=b_{i}\) as well as a linear transformation \(g: V \rightarrow F^{n}\) such that \(g\left(b_{i}\right)=e_{i}\text{.}\) Then both \(f \circ g: V \rightarrow V\) and \(g \circ f: F^{n} \rightarrow F^{n}\) are linear transformation which agree with the identity map on a basis. Hence by the uniqueness part of UMP for Free \(R\)-Modules we have \(f \circ g=\mathrm{id}_{V}\) and \(g \circ f=\mathrm{id}_{F^{n}}\text{.}\) Therefore, these maps are the desired isomorphisms.
  2. Let \(\varphi: F^{m} \cong F^{n}\) be a vector space isomorphism and let \(B\) be a basis of \(F^{m}\text{.}\) We claim that \(\varphi(B)\) is a basis for \(F^{n}\text{.}\) Indeed, if
    \begin{equation*} \sum_{i=1}^{m} c_{i} \varphi\left(b_{i}\right)=0 \quad \text { then } \varphi\left(\sum_{i=1}^{m} c_{i} b_{i}\right)=0, \quad \text { so } \quad \sum_{i=1}^{m} c_{i} b_{i}=0 \end{equation*}
    since \(\varphi\) is injective. But \(B\) is linearly independent, so we must have \(c_{i}=0\) for all \(1 \leqslant i \leqslant m\text{.}\) If \(v \in F^{n}\text{,}\) then since \(B\) spans \(F^{m}\) we have
    \begin{equation*} \varphi^{-1}(v)=\sum_{i=1}^{m} c_{i} b_{i} \end{equation*}
    for some \(c_{i}\text{.}\) Thus
    \begin{equation*} v=\sum_{i=1}^{m} c_{i} \varphi\left(b_{i}\right) \end{equation*}
    which shows \(\varphi(B)\) spans \(F^{n}\text{.}\) By the Dimension Theorem, we have
    \begin{equation*} \operatorname{dim}_{F}\left(F^{n}\right)=n=|\varphi(B)|=|B|=m. \end{equation*}
