Item 8.6.3. Endomorphism Ring.
If \(A = (A, +)\) is any abelian group, set \(\operatorname{End}_{Ab}(A)\) to be the collection of endomorphisms of \(A\) â€” that is, the set of group homomorphisms \(f: A \to A\) from \(A\) to itself.
Then \(\operatorname{End}_{Ab}(A)\) is a ring with addition \((f + g)(a) := f(a) + g(a)\) and multiplication \(f \cdot g := f \circ g\text{.}\) This is almost always a non-commutative ring.
